Sesión 1

En la primera sesión de Laboratorio de Innovación, hablamos sobre lo que hicimos en el primer laboratorio, donde usamos Arduino y aprendimos a programar. También discutimos sobre diseño web, como los navegadores, servidores web y dispositivos. Aprendimos sobre HTML, que es para hacer la estructura de una página web, y CSS, que es para darle estilo. Hablamos sobre la estructura básica de un elemento HTML y elementos importantes como el doctype, el elemento raíz y el elemento head. Fue una introducción al diseño web sin complicar las cosas.

Next

Sesión 2

En esta sesión te hablaré un poco sobre que el elemento "title" establece el título de tu página y el elemento "body" contiene lo que deseas mostrar a los usuarios. HTML ofrece seis niveles de encabezados, aunque generalmente se usan 3-4. Los elementos de contenido como "p" para párrafos, "ul" para listas no ordenadas y "ol" para listas ordenadas, junto con "li" para elementos de lista. Los vínculos o enlaces, implementados con el elemento "a" y para agregar imágenes, se utiliza el atributo src para especificar la ruta de la imagen. Características especiales de los elementos HTML incluyen el anidamiento y elementos vacíos como "img", que no necesitan etiqueta de cierre.

Next

Sesión 3

En nuestra última sesión, profundizamos en las ODS y su influencia en un futuro. Además, seguimos con Visual Studio Code, entendiendo su capacidad para manejar múltiples lenguajes de programación. No sólo eso, sino que también repasamos etiquetas HTML clave como "b", "i", "sup", "sub", "br" y "hr", y su función en nuestro código. Finalmente, analizamos la diferencia entre la sintaxis y la semántica, la sintaxis en programación se refiere a las reglas que rigen la estructura de un código y la semántica, por otro lado, trata sobre el significado de ese código. Ambos son esenciales para escribir y comprender programas correctamente.

Next

Sesión 4

En esta clase, tuvimos una sesión interesante donde todos mostraron sus páginas web para recibir retroalimentación del profesor. Cada uno presentó su bitácora y también el profesor nos guió mediante ejemplos. La personalización de las páginas web fue una parte destacada de la clase, ya que cada uno de mis compañeros hizo modificaciones según su gusto. Pasamos el resto del tiempo resolviendo dudas, especialmente en lo que respecta a la edición de páginas web a través de Visual Studio Code. El profesor fue de gran ayuda en este proceso de seguir mejorando el diseño de nuestra página web.

Next

Sesión 5

En esta sesión hablamos sobre las etiquetas HTML y sus usos: "strong" y "bold"que dan formato en negrita; "em> y "i" dan formato en cursiva al contenido; "blockquote" se usa para citas largas, puede generar una sangría en HTML sin CSS; "abbr" se usa para abreviaturas y acrónimos, lleva un atributo de título para especificar la palabra completa; "cite" se usa para hacer referencia a libros, grabaciones, artículos o documentos; se habló sobre la anidación de listas y los atributos en HTML, especificando los atributos para imágenes. Finalmente, se discutió sobre la inserción de videos Flash, MP4 y alternativos en una página web.

Next

Sesión 6

En la última clase, aprendí sobre HTML y CSS, dos lenguajes de programación fundamentales para el diseño web. El profesor explicó cómo el código HTML se organiza en "cajas" o "boxes", que incluyen elementos como el cuerpo del documento (body), los encabezados (h1) y los párrafos. En CSS, aprendí sobre la importancia de usar correctamente las llaves, los dos puntos y los puntos y comas. También hablamos sobre las etiquetas y los atributos para vincular correctamente los archivos CSS y HTML. Aprendimos cómo estilizar elementos específicos en CSS, como cambiar el color y el tamaño de la fuente.

Next

Sesión 7

Hoy en clase, aprendimos sobre la manipulación de elementos en CSS usando el modelo de cajas para ajustar dimensiones como altura, ancho, bordes, márgenes y rellenos. Discutimos sobre la interacción con el mouse, y cómo se pueden modificar los estilos de los enlaces y botones según si han sido visitados, si se está interactuando con ellos o si se está haciendo clic en ellos. En la sección de tablas, vimos cómo organizar y mostrar las celdas vacías y su espaciado. También, tipos de posicionamiento en CSS para ordenar cajas, incluyendo estático, relativo, absoluto, fijo y flotante. Vimos cómo usar float para ordenar elementos en múltiples columnas. Finalmente, tocamos el tema de hosting y dominio, discutiendo sobre sus componentes y algunas recomendaciones.

Next

Sesión 8

En mi última clase, aprendí a subir la carpeta de mi página web a GitHub, una plataforma de control de versiones que permite alojar y compartir mi código con otros. Este proceso me permitió crear un repositorio en línea para mi proyecto. Luego, usamos Netlify, una plataforma de despliegue y alojamiento web que se integra perfectamente con GitHub. Al vincular mi repositorio de GitHub con Netlify, pude generar un enlace único para mi página web, permitiendo a otros acceder y visualizar mi proyecto en línea de manera sencilla. Estas herramientas combinadas facilitaron el despliegue y la distribución de mi página web.
